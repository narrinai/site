Frequently Asked Questions

Everything you need to know about importing memories from ChatGPT

How do I get my ChatGPT memories?

Go to ChatGPT, paste the prompt above, and it will generate a JSON with your memories. Copy the entire response and paste it in the text area.

What information gets imported?

Personal information, preferences, goals, relationship details, and important conversation context that ChatGPT remembers about you.

Is my imported data secure?

Yes, all imported memories are encrypted and stored securely. Only you can access your imported memories, and they're never shared with other users.

Can I edit memories after importing?

Currently imported memories cannot be edited directly. However, you can always retake the personal quiz to update your preferences and add new information.

What if the import fails?

Make sure you're copying the complete ChatGPT response including the JSON format. If issues persist, try refreshing the page or contact support.

How many memories can I import?

There's no limit to the number of memories you can import. The system will process all valid memories from your ChatGPT export.
